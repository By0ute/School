# TIRF - Barcode Detection

## Coded with
* C++
* [**OpenCV**](http://opencv.org/)
* CMake

## Configuration
* Tested on Ubuntu 12.04

## How to Compile ?
```
cd TIRF/
mkdir build
cd build/
cmake ..
make
```

## Binary TirfBarCode
```
cd TIFG/
cd build/
./TirfBarCode <path-to-picture.png>
```
