# TIRF - Barcode Detection

## Coded with
* C++
* [**OpenCV**](http://opencv.org/)
* [**CMake**](http://www.cmake.org/)

## Configuration
* Tested on [**Ubuntu 12.04**](http://www.ubuntu.com/)

## How to Compile ?
```
cd TIRF/
mkdir build
cd build/
cmake ..
make
```

## Binary TirfBarCode
```
cd TIFG/
cd build/
./TirfBarCode <path-to-picture.png>
```
